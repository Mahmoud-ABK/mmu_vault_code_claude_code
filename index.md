# Claude Code Essence

A progressive 9-level guide to Claude Code, covering the full system from first session basics to autonomous multi-agent workflows. Each level builds on the previous, moving from context management and permissions through hooks, MCP servers, and orchestrated parallel agents.

the goal is to make most of Claude Code agentic loop , independently from the background models , whether Athropic or other models

## Structure

```
claude code/
  L1-First-Session.md                   # launch, interaction model, slash commands
  L2-Context-CLAUDE-md.md               # context window mechanics, project CLAUDE.md
  L3-Permissions-Settings.md            # allow/deny rules, project settings.json
  L4-Scoping.md                         # project vs global scope, ~/.claude/ layout, L5–L9 map
  L5-Hooks.md                           # lifecycle events, hook scripts
  L6-MCP-Servers-and-Plugins.md         # protocol, server config, plugins, LSP
  L7-Skills-Agents-Commands.md          # skills, sub-agents, custom commands
  L8-MultiAgent-Headless-Power.md       # worktrees, CI, scheduling, power prompting
  L9-Context-Cost.md                    # token budget of harness, global vs local cost
```
